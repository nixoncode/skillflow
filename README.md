# Skillflow

Skillflow is the backend API for a modern online learning platform. It provides the necessary infrastructure for managing users, courses, enrollments, and tracking student progress.

## Tech Stack

- **Backend:** Go (Golang)
- **Database:** MySQL
- **Routing:** echo framework
- **Authentication:** JWT

## Core Features

- User Registration and Authentication (Students & Instructors)
- Course & Lesson Management
- Student Enrollment System
- Progress Tracking

## Getting Started

All backend code and detailed setup instructions are located in the `/api` directory.

To get started, please refer to the [api/README.md](api/README.md) for a comprehensive guide on installation, configuration, and running the project.
